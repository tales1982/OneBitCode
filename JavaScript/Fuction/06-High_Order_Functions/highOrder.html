High Order Function

O que são as Higher Order Functions? (aka HOF).

Higher Order Function é toda função que aceita uma função como parâmetro e/ou retorna uma função...

  1. Torna o nosso código mais declarativo e de fácil leitura.
  2. É o conceito chave para conceitos mais complexos da programação, como closures, currying, promises, etc.
  3. Conseguimos generalizar funções para que possamos reutiliza-las de maneiras diferentes quando necessário.
