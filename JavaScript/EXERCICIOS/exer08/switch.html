Financial control

Write a javascript program that starts by asking for the amount initial amount of money available and then display that amount on the screen along with options to add and remove money and an option to go out. When clicking on the add money option, the program should ask by the amount to be added, add this value to the amount initial and then show again the menu with the amount of money and the options. The option to remove money should do the same, however subtracting the value. The exit option should terminate the program.
