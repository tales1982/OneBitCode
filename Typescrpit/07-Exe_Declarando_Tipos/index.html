EXERCICIOS DE TYPESCRIPT

  1. A primeira funcao devera salva um objeto spaceShip com, no minimo, as seguites propiedades:

    Name que devera ser inseriada pelo usuario.
    Pilot que devera ser inserido pelo usuario.
    CrewLimit o tamanho maximo da tripulacao que devera ser inserido pelo usuario.
    Crew um array de string inicialmente vazio.
    InMisision que inicalmente deve ser falso.

  2. A segunda funçao deverá adcionar um membro à tripulaçao de uma determinada nave.
    No entanto o tamanho maximo nao pode ser excedido.

  3. A terçeira funçao deverá enviar uma determinada nave em uma missao caso ela ja nao esteja.
    Para isso é precico que 1/3 da sua tripulaçao esteja preenchida, arredondando pra baixo.

  4. A quarta funçao deverá lista todas as naves registradas e suas respectivas informacoes.
